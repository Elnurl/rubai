Title: Implement “Momentum + Adaptive Daily Task System + Real Image Attachments”
We are upgrading an AI productivity mobile/web app. Implement the following features inside the existing system.
________________________________________
1. Momentum / Risk Status (Today Screen)
Create a dynamic “Momentum Status” component on the Today screen.
Goal:
Show a simple behavioral insight message based on user activity patterns.
Inputs (already available in system):
•	daily task completion rate 
•	streak length 
•	recent activity (last 3–7 days) 
•	engagement frequency 
Output:
A single dynamic status message such as:
•	“You’re losing momentum — let’s get back on track.” 
•	“Great consistency — keep it going.” 
•	“High performance week — strong discipline detected.” 
________________________________________
2. AI Adaptive Daily Task System (Core Upgrade)
This is the most important feature.
Goal:
Every day, AI must generate a personalized task plan based on:
•	previous day performance 
•	user behavior patterns 
•	cognitive load (task difficulty balance) 
•	user preference history 
Behavior:
At the start of each day:
1.	Analyze previous day: 
o	completed tasks 
o	skipped tasks 
o	timing behavior 
2.	Identify pattern: 
o	productivity peak time 
o	weak areas 
o	overload signs 
3.	Generate next-day tasks: 
o	adjust difficulty dynamically 
o	balance workload (not too easy, not too hard) 
o	continue unfinished tasks if relevant 
o	introduce new tasks if user is stable 
________________________________________
